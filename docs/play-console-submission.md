## 4. 카테고리 / 태그

- 카테고리: **소셜 > 데이트** (Social > Dating)
- 주요 키워드: `시니어매칭`, `중년소개팅`, `40대만남`, `50대만남`, `60대만남`, `가치관매칭`, `재혼`, `황혼`, `동반자`

## 5. 콘텐츠 등급

데이팅 카테고리는 자동으로 **청소년 이용 불가 (만 18세 이상)** 으로 분류됩니다. 추가로 IARC 설문에서 아래와 같이 답변:

- 폭력성: 없음
- 성적 콘텐츠: 암시적 (데이팅 앱 특성상)
- 욕설: 없음 (금칙어 필터 적용)
- 규제 콘텐츠: 없음 (도박·마약 없음)
- 이용자 상호작용: **있음** (메시지, 프로필 공유)
- 위치 공유: 없음 (시/도 단위만, GPS 미사용)

## 6. 데이터 보안 설문 (Data Safety) — Play Console 필수

Play Console → 정책 → 앱 콘텐츠 → 데이터 보안

### 수집/공유 개요
- 수집: **예**
- 제3자 공유: **아니요** (수사기관 협조·매칭 상대에게 표시되는 프로필은 법령·사용자 동의 하의 제공)
- 암호화 전송: **예** (HTTPS 필수, cleartext 차단)
- 데이터 삭제 요청: **앱 내 "계정 삭제" 즉시 삭제 가능**
