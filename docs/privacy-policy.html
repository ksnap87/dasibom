4. 개인정보의 제3자 제공

다시봄은 이용자의 개인정보를 원칙적으로 외부에 제공하지 않습니다. 다만, 아래의 경우에 한해 예외로 합니다:

  • 이용자가 사전에 동의한 경우
  • 법령에 의해 요구되는 경우 (수사 기관의 적법한 요청 등)

매칭된 상대방에게는 이용자가 직접 입력한 프로필 정보(이름, 출생연도, 거주 지역, 가치관 설문 응답, 프로필 사진)가 표시됩니다. 휴대전화번호, 카카오 계정 정보 등 민감한 개인정보는 상대방에게 제공되지 않습니다.

6. 개인정보의 안전성 확보 조치

다시봄은 개인정보의 안전성 확보를 위해 다음과 같은 조치를 취하고 있습니다:

  • 암호화: 휴대전화번호는 AES-256-GCM으로 암호화하여 저장하며, 비밀번호는 단방향 해시 처리합니다.
  • 전송 구간 보안: 모든 통신은 HTTPS(TLS 1.2 이상)를 통해 암호화됩니다.
  • 접근 제한: 개인정보에 대한 접근 권한을 최소한으로 제한하며, 데이터베이스 행 단위 보안(RLS)을 적용합니다.
  • 금칙어 필터링: 채팅 메시지에서 부적절한 내용을 자동 감지하여 안전한 이용 환경을 유지합니다.

7. 이용자의 권리 및 행사 방법

이용자는 언제든지 다음의 권리를 행사할 수 있습니다:

  • 열람 요구: 자신의 개인정보 처리 현황 확인
  • 정정 요구: 부정확한 개인정보의 수정 요청
  • 삭제 요구: 앱 내 "계정 삭제" 기능으로 즉시 삭제 가능 (프로필 > 설정 > 계정 삭제)
  • 처리 정지 요구: 개인정보 처리의 일시 정지 요청
  • 동의 철회: 앱 내 설정 또는 이메일을 통해 동의 철회 가능

권리 행사는 앱 내 설정 또는 아래 연락처를 통해 가능하며, 요청 접수 후 10일 이내에 처리합니다.

8. 만 14세 미만 아동의 개인정보

다시봄은 만 50세 이상 시니어를 대상으로 하는 서비스로, 만 14세 미만 아동의 개인정보를 수집하지 않습니다. 만 14세 미만 아동의 가입이 확인될 경우 즉시 계정을 삭제합니다.
