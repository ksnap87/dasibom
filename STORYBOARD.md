# 다시봄 — 사용자 스토리보드

## 1. 첫 진입 (스플래시)
```
┌─────────────────────┐
│                     │
│     🌸 (fade-in)    │
│                     │
│      다시봄          │
│  다시 찾는 봄날의 인연  │
│                     │
│  ● ● ● (로딩)       │
│                     │
└─────────────────────┘
         │ 2초 후
         ▼
```

## 4. 추천 탭 (메인)
```
┌─────────────────────┐
│  🌸 다시봄 추천       │
├─────────────────────┤
│  ┌─────────────────┐│
│  │  📷 프로필 사진    ││
│  │                 ││
│  │  김영희, 55세     ││
│  │  서울 · 92점 💕   ││
│  │                 ││
│  │  Q&A 미리보기:    ││
│  │  · 종교: 불교     ││
│  │  · 목표: 재혼     ││
│  │  · 운동: 주 3회   ││
│  └─────────────────┘│
│                     │
│  [👎 패스]  [💌 관심] │
│     ← 스와이프 →      │
├─────────────────────┤
│  🌸추천  💌채팅  👤프로필│
└─────────────────────┘

로딩 시: SkeletonLoader (카드형)
빈 상태: "오늘의 추천을 모두 확인했어요!"
좋아요 시: Toast "관심을 표현했어요 💌"
```
